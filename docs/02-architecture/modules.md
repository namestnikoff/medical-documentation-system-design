# Модули и границы ответственности

| Модуль                      | Ответственность                                  |
|-----------------------------|--------------------------------------------------|
| Пациенты                    | CRUD, поиск, история болезни                     |
| Расписания                  | График работы и слоты для записи                 |
| Приёмы                      | Оформление приёма и его связи с документами      |
| Медицинские документы       | Формирование, подписание, отправка               |
| Пользователи                | Роли, права, аутентификация                     |
| Платные услуги              | Учёт, оплата, договора ДМС                      |
| Статистика                  | Автоматизация отчётов, выгрузка форм             |
| Аудит                       | Логирование действий                            |
| Интеграция                  | ЕГИСЗ, ЕСИА, справочники                        |
