# Типовой сквозной сценарий

**Сценарий: Оформление приёма и формирование СЭМД**

1. Пациент записывается на приём через личный кабинет/регистратуру.
2. Врач открывает приём, вводит жалобы, проводит осмотр, ставит диагноз, назначает лечение.
3. Врач формирует медицинский документ на основе приёма.
4. Врач подписывает документ электронной подписью.
5. Документ автоматически сохраняется и уходит в ЕГИСЗ (если требуется).
6. Пациент может просмотреть приём и назначение в своём личном кабинете.
